# [ hodo ](https://github.com/bernadetteabalos/hodo-app)
## A collaborative whiteboard application that allows users to get creative while brainstorming or planning.


_Hodo was created as part of the final project at Lighthouse Labs, and presented during Demo Day 2022._

### Tech stack:

- Front-end: HTML, CSS, React, KonvaJS
- Back-end: Node, Express, SQL, Socket.IO
